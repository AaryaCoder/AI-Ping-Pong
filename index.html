AI PING PONG GAME











Instructions

  1. 1. Place your laptop in a comfortable position
  2. 2. Stay 3 ft away from screen
  3. 3. Click the play button to start the game
  4. 4. Lift up your right wrist so it can be seen by the camera. Move it up and down to control the red paddle
  5. 5. Use the restart button if you want to play again
